# KessDB Site

This repository contains the public-facing KessDB whitepaper site.

KessDB itself is proprietary, private-source software owned by Jason Thompson. The product source repository is separate from this public site repository.

## GitHub Pages

Recommended Pages settings:

- Repository: `Metalios/KessDB-site`
- Visibility: public
- Source: deploy from a branch
- Branch: `main`
- Folder: `/`

The site entry point is `index.html`.
