# KessDB Site

This repository contains the public-facing KessDB whitepaper site.

KessDB itself is proprietary, source-available software owned by Jason Thompson. The product source repository should remain separate from this public site repository.

## GitHub Pages

Recommended Pages settings:

- Repository: `Metalios/KessDB-site`
- Visibility: public
- Source: deploy from a branch
- Branch: `main`
- Folder: `/`

The site entry point is `index.html`.
